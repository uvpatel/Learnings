HTML Lists

Lists are used to display information in a structured format. There are three types of lists in HTML:

            <ul> - Unordered List
            <ol> - Ordered List
            <dl> - Definition List
            <li> - List Item
            <dt> - Definition Term
            <dd> - Definition Description
